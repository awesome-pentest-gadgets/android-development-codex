    Ultimate Tutorial to Install 6.XX PRO CFW for all PSP models

    pro-c is on the homepage of the procfw site!!

    http://code.google.com/p/procfw/

    linkbucks and sites like them that generate money from people going to them ARE NOT ALLOWED ON THIS SITE! —Hacks Forum Rules at the time this tut was edited the links were in a normal spot on the downloads page..I nor anyone here knows why the PRO team decided to move them let alone put them on linkbucks so dont ask!!

    This tutorial covers installing the latest 6.XX PRO series custom firmware for all PSP models. I have tried to include all firmware/psp model combinations to help you decide which path to follow.


    Requirement

    - Any PSP in working condition
    - Motherboard version in case you have a PSP 2000
    - Module number in case you have a PSP 3000

    Identify the motherboard for PSP 2000

    Spoiler for Identify the motherboard:
    The motherboard can be identified on all HENs/CFWs using an app called PSPIdent, but for the sake of consistency lets use 6.39 PRO B-10 and PSPIdent to identify the motherboard.

    1. Download OFW 6.39 - google :) all the firmwares of sonys have been removed from this site if you want to know more go here Why can't I find OFW updates anymore?
    2. Extract the file
    3. Copy EBOOT.PBP to PSP/GAME/UPDATE/
    4. From the xmb (main psp menu) run the update from Game – Memory Stick
    5. Now you are on OFW 6.39
    6. Download 6.39 PRO-C fix4 - 639PRO-C.fix4.rar - procfw - 639PRO-C fix4 - X-Generation Aftermarket Firmware for Playstation Portable
    7. Extract the file
    8. Copy PROUPDATE and FastRecovery to PSP/GAME/
    9. From the xmb (main psp menu) run the PROUPDATE from Game - Memory stick.
    10. Press X to install CFW
    11. Download PSPident - PSPident 0.74.2 | PSP-Hacks: Hack your PlayStation Portable
    12. Extract the file
    13. Copy PSPIDENT folder to PSP/GAME/
    14. From the xmb (main psp menu) run PSPIDENT
    15. Make note of the motherboard model and follow the appropriate guide to install CFW


    Identify the module PSP 3000
    Spoiler for Identify the module PSP 3000:
    Identify the module of your psp using this guide (credits to kamenrider)
    http://forums.dashhacks.com/f133/the...st1557692.html


    PSP 1000 – Install 6.60 PRO B-10

    • If Official Firmware is below 6.60

    Spoiler for Instructions If Official Firmware is below 6.60:
    1. Download OFW 6.60 - google :) or if connected to wifi then network update all the firmwares of sonys have been removed from this site if you want to know more go here Why can't I find OFW updates anymore?
    2. Extract the file
    3. Copy EBOOT.PBP to PSP/GAME/UPDATE/
    4. From the xmb (main psp menu) run the update from Game – Memory Stick
    5. Now you are on OFW 6.60
    6. Download 6.60 PRO-C fix3 - 660PRO-C.fix3.rar - procfw - 660PRO-C fix3 - X-Generation Aftermarket Firmware for Playstation Portable
    7. Extract the file
    8. Copy PROUPDATE and CIPL Flasher folders to PSP/GAME/
    9. From the xmb (main psp menu) run the PROUPDATE from Game - Memory stick.
    10. Press X to install CFW
    11. From the xmb (main psp menu) run the CIPL Flasher from Game - Memory stick.
    12. You have successfully installed 6.60 PRO-C fix3
